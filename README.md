# Basic Rule-Based Chatbot

## 📌 Description
This is a simple rule-based chatbot built using Python. It responds to predefined user inputs.

## 🚀 Features
- Responds to:
  - "hello"
  - "how are you"
  - "bye"
- Uses if-elif conditions, functions, and loops

## 🛠 Requirements
- Python 3.x

## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/your-username/basic-chatbot.git

2. Navigate to the folder:
   cd basic-chatbot

3. Run the chatbot:
   python chatbot.py

## 💡 Example
You: hello  
Bot: Hi!

You: bye  
Bot: Goodbye!

## 📚 Concepts Used
- Functions
- Loops
- Conditional statements (if-elif)
- Input/Output
